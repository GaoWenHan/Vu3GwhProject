# Vue 3 + TypeScript 项目模板

## 特性
- Vue 3 组合式API
- TypeScript支持
- Webpack构建
- ESLint + Prettier代码规范
- Vitest单元测试
- Tailwind CSS
- 预配置Git Hooks

## 快速开始

1. 使用pnpm创建项目：
```bash
pnpm create vue-ts-template my-project
```

2. 安装依赖：
```bash
cd my-project && pnpm install
```

3. 开发模式：
```bash
pnpm dev
```

4. 生产构建：
```bash
pnpm build
```
